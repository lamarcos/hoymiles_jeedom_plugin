# Changelog Hoymiles

## Version 1.0.0 (Date initiale)

### Fonctionnalités

- Première version du plugin
- Connexion à l'API Hoymiles
- Récupération des données de production solaire
- Publication des données vers MQTT
- Support des métriques :
  - Puissance instantanée
  - Énergie journalière
  - Énergie totale
  - Énergie mensuelle
  - Énergie annuelle
- Daemon Python pour polling régulier
- Configuration complète via l'interface Jeedom
- Support authentification MQTT (optionnelle)
- Intervalle de polling configurable
